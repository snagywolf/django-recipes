# Django Recipes

Sistema de receitas desenvolvido com Django + Django REST Framework.

## Funcionalidades

- Cadastro e login de usuários
- Dashboard com receitas
- Criar, editar e deletar receitas
- Aprovação por administrador (is_published)
- API REST com DRF
- Autenticação JWT

## Tecnologias

- Python
- Django
- Django REST Framework
- SQLite
- JWT (SimpleJWT)

## Como rodar o projeto

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
